# 인스타 카드 — 텍스트 입력 규칙 (메모리)

> PresenceWorld 인스타 카드용. 제품 사진 1장을 받으면 이 규칙대로
> **텍스트 4줄**을 뽑아 사용자에게 전달한다. 사용자가 도구의 editable
> 필드에 붙여넣고 2차 수정한다. (사진엔 글자가 없으므로 OCR이 아니라
> 이미지 인식으로 캐릭터·IP를 판별한다.)

## 출력 형식 (항상 이 4줄, 이 순서)

```
헤드라인 : <공개/판매 상태>
타이틀   : <작품 한글명> <시리즈명>
제품명   : <캐릭터 한글명> (<버전/상태>)
푸터     : 메가하우스 공식 스토어
```

## 필드별 규칙

### ① 헤드라인 — 공개/판매 상태
사진·맥락으로 판단. 기본 분류:
- `원형 최초공개` — 미채색(원형) 시제품 첫 공개
- `채색원형 최초공개` — 채색 시제품 첫 공개
- `재판매 결정` — 재판/재입고
- (확장) `예약 시작`, `발매 결정`, `한정 재판매 결정`
- 판단 불가하면 `원형 최초공개` 로 두고 사용자가 고치게 한다.

### ② 타이틀 — "작품 한글명 + 시리즈명"
- 형식: `{작품 정식 한글명} {시리즈명}`
- 예: `나루토 질풍전 G.E.M. 시리즈`, `진격의 거인 G.E.M. 시리즈`, `은혼 G.E.M. 시리즈`
- 작품명은 **한국 정식 발매 명칭** 우선. 없으면 통용 음역.
- 시리즈명 = 아래 **제품 라인 분류**에서 판별한 라인명. (기본값 없음 — 사진으로 분류)
- **접미 규칙 (이것만):**
  - `G.E.M.` → `G.E.M. 시리즈` ('시리즈' 붙임)
  - `메가캣` → `메가캣 프로젝트` ('프로젝트' 붙임)
  - 그 외 전부 **접미 없이 라인명 그대로**:
    `룩업`, `스케일 피규어`, `컬렉션 피규어`, `컬렉션 굿즈`, `GGG`, `루크리아`, `멜티 프린세스`
- 예: `주술회전 룩업`, `진격의 거인 스케일 피규어`, `나루토 질풍전 G.E.M. 시리즈`, `원피스 메가캣 프로젝트`

### ③ 제품명 — 캐릭터 한글명 (+ 버전/상태)
- 캐릭터 한글명은 **한국 정식 표기** 우선.
- 여러 명이면 `&` 로 연결: `에렌 예거 & 리바이 병장`
- 버전·부가는 괄호로, 원문 뉘앙스 유지:
  `(미채색)`, `(유년 Ver.)`, `(표정 파츠 동봉)`, `ver.弐 이 (발차기)`
- 세부 라인 접두어가 있으면 캐릭터 앞에 붙인다: `테노히라 나미카제 미나토`

### ④ 푸터 — 고정
- `메가하우스 공식 스토어` (다른 브랜드면 교체)

## 제품 라인(시리즈) 분류 — 사진으로 먼저 판별

인식 순서: **(1) 어느 라인인가 → (2) 어느 작품/캐릭터인가 → (3) 4줄 작성.**
라인명은 ②타이틀의 시리즈명, 또는 ③제품명 접두어로 들어간다.

### A. 룩업 (LookUp) — 위를 올려다보는 데포르메
- `룩업` (루카푸) — 기본 룩업 피규어
- `룩업 의상` (루카푸 폰초) — 룩업 전용 의상·소품
- `룩업 미니어처` — 룩업을 키링 사이즈로 작게
- `룩업 소품` — 룩업 전용 파우치 등 소품류
- 인식단서: 고개를 위로 든 둥근 데포르메.

### B. 스케일 피규어 (1차분류)
정밀 비율 스케일 제품군. 아래 **2차분류**로 나뉜다 → 타이틀엔 2차분류명이 들어간다.
- **G.E.M. 시리즈** — 타이틀엔 항상 `G.E.M. 시리즈`. 내부 변형은 **제품명 접두어**로:
  - `그냥 G.E.M.` / `프레셔스 G.E.M.`(Precious) / `G.E.M. 테노히라`(손바닥 미니)
  - 예) 타이틀 `나루토 질풍전 G.E.M. 시리즈` + 제품명 `테노히라 나미카제 미나토`
- **GGG** / **루크리아**(Lucrea) / **멜티 프린세스**(Melty Princess/Prince) — 타이틀엔 그 이름 그대로
- **분류없음** — 타이틀엔 `스케일 피규어` (예: '메가하우스 스케일 피규어 고죠 사토루')
- 인식단서: 정밀 비율 전신=스케일, 손바닥 위 미니 앉은형=`G.E.M. 테노히라`.

### C. 메가캣 프로젝트 (MEGA CAT PROJECT) — 캐릭터 고양이화
- `메가캣` — 기본 메가캣 (n종 트레이딩)
- `빅 메가캣` — 초대형 단일 제품
- `메가캣 잘자` — 병 안에 가둔 형태
- `메가캣 찰싹` — 폰에 붙이는 그립톡 형태
- `메가캣 뱃지` — 일러스트 뱃지 굿즈
- 인식단서: 캐릭터가 고양이 몸/귀로 변형.

### D. 컬렉션 / 트레이딩 피규어
- `쵸코링 컬렉션` — 미니 데포르메 피규어 시리즈
- `기타 컬렉션` — 로그박스(LOGBOX), 츠미첸, 오챠토모 등 소형 피규어·박스 굿즈

### E. 컬렉션 / 트레이딩 굿즈 (피규어 아님)
- `카라코로` — '칼라풀 코롯토', 색감 예쁜 아크릴 코롯토
- `버디코레` — '버디 콜렉션', 2인 페어 일러스트 굿즈
- `이루스타` — 'いる(있다)+스탠드', 색감이 강점인 LD 아크릴 스탠드
- `꺄르르르` — 크레인 가챠 컨셉의 아크릴 스탠드
- 인식단서: 입체 피규어가 아닌 평면 아크릴/굿즈.

## 원문 → 정형 입력 매핑 (도구가 양식을 고정함)

도구는 타이틀을 자유입력하지 않고 **[헤드라인 드롭다운] + [작품명] + [라인 드롭다운]**으로
받아 `작품 + 라인` 순서로 자동 조립한다. 원문을 받으면 아래 형태로 정리해 준다:

```
헤드라인 : <드롭다운 값 중 하나>
작품명   : <IP 한글명>          (예: 은혼)
라인     : <드롭다운 값 중 하나> (예: G.E.M. 시리즈)
제품명   : <캐릭터, 여러 명은 줄바꿈>
일본어   : <해시태그용 원작명>   (예: 銀魂)
```

## 원고(캡션) 형식 — 도구가 자동 생성

```
📢[{헤드라인}] {작품} {라인}

{제품명 줄1}
{제품명 줄2}

※ 예약 오픈 시 별도 공지
※ 일본 현지 기준 소식으로, 국내외 사정에 따라 일부 사항이 변동될 수 있습니다.

#{일본어원작} #{작품} #메가하우스공식스토어
```

- 헤드라인이 `한정 재판매 결정`이면 제품명 각 줄 끝에 `【한정 재판매판】`을 붙인다.
- 일본어 해시태그는 원작 일본어명(없으면 생략).

## 인식 시 주의
- **라인 분류 우선.** 라인이 애매하면 후보를 같이 적는다.
- 확신이 낮은 캐릭터/작품은 **추정임을 표시**하고 후보를 제시한다.
- 절대 헛값을 단정하지 않는다 — 모르면 "확인 필요"로 남기고 사용자가 채우게 한다.
- 출력은 곧바로 붙여넣을 수 있게 4줄만, 군더더기 없이.
