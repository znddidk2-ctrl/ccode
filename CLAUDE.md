# 인스타 카드 — 텍스트 입력 규칙 (메모리)

> PresenceWorld 인스타 카드용. 제품 사진 1장을 받으면 이 규칙대로
> **텍스트 4줄**을 뽑아 사용자에게 전달한다. 사용자가 도구의 editable
> 필드에 붙여넣고 2차 수정한다. (사진엔 글자가 없으므로 OCR이 아니라
> 이미지 인식으로 캐릭터·IP를 판별한다.)

## 출력 형식 (항상 이 4줄, 이 순서)

```
헤드라인 : <공개/판매 상태>
타이틀   : <작품 한글명> <시리즈명>
제품명   : <캐릭터 한글명> (<버전/상태>)
푸터     : 메가하우스 공식 스토어
```

## 필드별 규칙

### ① 헤드라인 — 공개/판매 상태
사진·맥락으로 판단. 기본 분류:
- `원형 최초공개` — 미채색(원형) 시제품 첫 공개
- `채색원형 최초공개` — 채색 시제품 첫 공개
- `재판매 결정` — 재판/재입고
- (확장) `예약 시작`, `발매 결정`, `한정 재판매 결정`
- 판단 불가하면 `원형 최초공개` 로 두고 사용자가 고치게 한다.

### ② 타이틀 — "작품 한글명 + 시리즈명"
- 형식: `{작품 정식 한글명} {시리즈명}`
- 예: `나루토 질풍전 G.E.M. 시리즈`, `진격의 거인 G.E.M. 시리즈`, `은혼 G.E.M. 시리즈`
- 작품명은 **한국 정식 발매 명칭** 우선. 없으면 통용 음역.
- 시리즈명 기본값: `G.E.M. 시리즈` (다른 라인이면 교체)

### ③ 제품명 — 캐릭터 한글명 (+ 버전/상태)
- 캐릭터 한글명은 **한국 정식 표기** 우선.
- 여러 명이면 `&` 로 연결: `에렌 예거 & 리바이 병장`
- 버전·부가는 괄호로, 원문 뉘앙스 유지:
  `(미채색)`, `(유년 Ver.)`, `(표정 파츠 동봉)`, `ver.弐 이 (발차기)`

### ④ 푸터 — 고정
- `메가하우스 공식 스토어` (다른 브랜드면 교체)

## 인식 시 주의
- 확신이 낮은 캐릭터/작품은 **추정임을 표시**하고 후보를 제시한다.
- 절대 헛값을 단정하지 않는다 — 모르면 "확인 필요"로 남기고 사용자가 채우게 한다.
- 출력은 곧바로 붙여넣을 수 있게 4줄만, 군더더기 없이.
