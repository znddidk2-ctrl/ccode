# 엑셀 생성 규칙

## 파일 명명 규칙
- raw 데이터: `{플랫폼}_raw.xlsx` (예: X_raw.xlsx, 인스타_raw.xlsx)
- 결과 통계: `통계_{월}.xlsx` (예: 통계_5월.xlsx)

## 시트 구성
1. **원본** — raw 데이터 그대로 붙여넣기
2. **정제** — 중복 제거, 빈 행 삭제, 날짜 포맷 통일 (YYYY-MM-DD)
3. **통계** — 집계 결과 (아래 항목 포함)

## 통계 항목 (플랫폼 공통)
| 항목 | 설명 |
|------|------|
| 총 게시물 수 | 해당 월 전체 |
| 총 노출(impression) | 합계 |
| 총 참여(engagement) | 좋아요+댓글+공유 합계 |
| 평균 참여율 | engagement / impression × 100 |
| 상위 5 게시물 | 참여수 기준 내림차순 |

## 전월 비교
- 전월 `통계_{월}.xlsx`의 **통계** 시트를 참조
- 증감률 = (당월 - 전월) / 전월 × 100, 소수점 1자리
- 증가 시 ▲, 감소 시 ▼ 표기

## 주의사항
- 인코딩: UTF-8
- 날짜 열은 텍스트가 아닌 날짜 형식으로 저장
- 수식은 값으로 붙여넣기 후 저장 (다른 환경 호환)
