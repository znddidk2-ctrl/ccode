# 리포트 작성 규칙

## 리포트 목적
메가하우스 SNS 채널(X·인스타그램)의 월간 성과를 정리하고 전월 대비 변화를 파악

## 파일 명명
- `리포트_{YYYY-MM}.md` (예: 리포트_2026-06.md)

## 구조

### 1. 요약 (Executive Summary)
- 3~5줄, 핵심 수치만 포함
- 예: "6월 인스타 노출 전월 대비 ▲12.3%, X 참여율 ▼2.1%"

### 2. 플랫폼별 성과

#### 2-1. X (구 트위터)
| 지표 | 당월 | 전월 | 증감 |
|------|------|------|------|
| 게시물 수 | | | |
| 총 노출 | | | |
| 총 참여 | | | |
| 평균 참여율 | | | |

#### 2-2. 인스타그램
(동일 표 구조)

### 3. 상위 콘텐츠 TOP 5
- 플랫폼 / 날짜 / 내용 요약 / 참여수 순으로 기재

### 4. 특이사항 및 인사이트
- 급상승·급하락 원인 추정
- 이벤트·캠페인 연계 여부

### 5. 다음 달 제언
- 2~3개 항목, 구체적인 액션 위주

## 작성 지침
- 수치는 `통계_{월}.xlsx`의 **통계** 시트 기준
- 전월 비교는 직전 월 폴더의 통계 파일 참조
- 숫자 단위: 1,000 이상은 천 단위 쉼표 사용
- 작성 후 `2026-{MM}/` 폴더에 저장
