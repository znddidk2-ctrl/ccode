# 인스타 카드 생성기

사진 **1장**을 올리면 인스타용 카드(1080×1350)를 만들어 주는 단일 웹 도구입니다.
`index.html` 하나로 동작 — 빌드/서버 없이 브라우저로 바로 엽니다.

## 레이어 구조 (3층)

```
1층  텍스트     ← editable 필드에 붙여넣고 수정
2층  사진       ← 업로드한 사진, 웹에서 크롭(드래그/줌)만 가능
3층  배경지     ← 같은 원본 이미지를 불투명 70% + 흐림으로 깔기
```

로고·한자·배지·누끼 **없음**. 폰트는 **Pretendard (300~800)** 로 통일.

## 텍스트는 어디서 오나 (중요)

업로드하는 건 **글자 없는 제품 사진**이라 OCR로는 캐릭터·IP를 못 읽는다.
대신 **Claude에게 사진을 주면 `CLAUDE.md` 규칙대로 텍스트 4줄을 뽑아준다.**
그걸 필드에 붙여넣고 틀린 곳만 고치면 끝. (텍스트 인식 규칙 = 레포 루트 `CLAUDE.md`)

## 사용법

1. 제품 사진을 Claude에게 줘서 텍스트 4줄(헤드라인/타이틀/제품명/푸터)을 받는다.
2. `index.html` 을 브라우저로 연다 (더블클릭).
3. 같은 사진 1장 업로드 → 배경지(3층)와 사진(2층)에 자동 적용.
4. 받은 텍스트를 필드에 붙여넣고 editable로 2차 수정.
5. 사진을 **드래그해서 위치 조정**, 휠/슬라이더로 **확대**(크롭).
6. 배경 불투명·흐림 슬라이더로 미세 조정.
7. **PNG 다운로드** 클릭 → `insta-card.png` 저장.

## 외부 의존 (CDN)

| 용도 | 출처 |
|---|---|
| Pretendard 폰트 | `cdn.jsdelivr.net/gh/orioncactus/pretendard` |

네트워크가 막히면 폰트는 시스템 폰트로 폴백된다. 완전 오프라인이 필요하면
폰트 파일을 로컬에 받아 경로만 바꾸면 된다. (그 외 기능은 전부 클라이언트에서 동작)

## 커스터마이즈 포인트 (index.html 내부)

- `FRAME` — 사진 프레임 위치/크기/모서리
- `render()` 의 `fillText` 좌표·`setFont(weight,size)` — 텍스트 배치/굵기
- 상/하 스크림 그라데이션 — 텍스트 가독성
